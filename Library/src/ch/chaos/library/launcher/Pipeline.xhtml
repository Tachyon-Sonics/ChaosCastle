Java2D Pipeline

Allows you to select the 2D rendering pipeline to use.

This setting currently has little effect because the game uses indexed (palette) colors, which is not hardware optimised. In fact, the game achieves smooth scrolling at 60 FPS even with the "Software" mode at a resolution of 1920x1080 and highest quality on modern laptops. This setting only affects the final on-screen rendering and the "Outer Scaling", if greater than 1.

A few caveats:

  • The available pipelines differ between operating systems.
  • "Default" is usually the safest one.
  • Some pipelines ("Direct3D" on Windows, "OpenGL" on Linux) may have trouble rendering in-game menus and dialogs on hardware that is not fully compatible, especially in full-screen mode. Workaround is to pause the game, return to the main screen with [ALT]-[TAB] as necessary and then use the menu from the tray icon.
