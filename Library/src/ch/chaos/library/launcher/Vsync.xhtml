V-Sync Mode

When hardware V-Sync (vertical sync) is not supported, this setting selects how to do it in software.

  • Sleep: use software sleeping. This mode has the lowest CPU consumption and the least drain on the battery; however it might drift from a few milliseconds and hence result in jaggy scrolling
  • Balanced low: use a dynamic combination of software sleeping and active waiting. This uses more CPU than Sleep, but significantly reduces jaggy scrolling.
  • Balanced high: same as Balanced low, with less software sleeping and more active waiting. Uses even more CPU, but almost completely reduce jaggy scrolling.
  • Active: use active wating only. In this mode the CPU is constantly used at 100%, and will drain the battery the most. It might be smoother than Balanced high in some cases.

Note that the chosen mode may have no impact at all on slow computers that cannot achieve 60 FPS at all.
