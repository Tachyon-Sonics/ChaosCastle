Use by default and do not ask again

When checked, this dialog will no longer open and the selected settings will always be used.

You can force displaying this launcher dialog again be starting the game with the "--settings" command-line argument, or by choosing the "Settings/Misc..." item from the in-game menu (use the right mouse button to open the in-game menu during the game).

If the display mode is modified at the operating system level, this dialog my show again even if this option was checked, because the settings are stored separately for each display mode.
